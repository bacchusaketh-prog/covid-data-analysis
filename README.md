\# COVID-19 Data Analysis using R



This project performs exploratory data analysis on COVID-19 patient data using R.



\## Dataset

COVID19\_line\_list\_data.csv containing patient information such as:

\- Age

\- Gender

\- Country

\- Death status

\- Recovery status



\## Analysis Performed

\- Age vs mortality statistical testing

\- Gender mortality comparison

\- Logistic regression model

\- Country-level case analysis

\- Global COVID case visualization



\## Visualizations

The project generates multiple visualizations including:

\- Age distribution

\- Death rate by gender

\- Top countries by COVID cases

\- Age group mortality

\- Correlation heatmap

\- Global COVID map

\- Daily case trend



\## Tools Used

\- R

\- ggplot2

\- dplyr

\- corrplot
